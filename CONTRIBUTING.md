# Contributing to n8n Workflow Manager

Thank you for your interest in contributing!

## Development Setup

```bash
git clone https://github.com/lukisch/n8n-workflow-manager.git
cd n8n-workflow-manager
pip install -e ".[dev]"
```

## Running Locally

```bash
python -m n8nManager serve
```

## Code Style

- Python: Follow PEP 8, use `ruff` for linting
- JavaScript: No framework, vanilla JS
- Comments: English

## Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit changes (`git commit -m "Add my feature"`)
4. Push to branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Reporting Issues

Use [GitHub Issues](https://github.com/lukisch/n8n-workflow-manager/issues) to report bugs or request features.
